# Agenda Eletronica TCC (MERN)

Backend em Express e MongoDB (Atlas) de uma agenda eletronica com clientes e comentarios. 
Frontend em React.

```
edit config.env
```

```
 # QUICK START
 npm install
 npm run install-client
 npm run dev
 
 # Run front and backend
 npm run dev
 
 # Backend only
 npm run server
 
 # Frontend only
 npm run client
 
 # Build client
 cd client
 npm run build
 
 ou
 npm run install-client
 
 # Prod
 npm start
```
